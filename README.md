# React-Redux-Pokedex

## USAGE
| command               | description                |
|:----------------------|:---------------------------|
| `$ yarn`              | installation               |
| `$ yarn start`        | server start               |
| `$ yarn build`        | build                      |
| `$ yarn test`         | test                       |
| `$ yarn flow`         | flow                       |
| `$ yarn flow-coverage`| to render coverage of flow.|
